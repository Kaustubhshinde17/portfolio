# 🚀 Premium Personal Portfolio Website

This is a premium, recruiter-focused personal portfolio website built with **React**, **Vite**, **Tailwind CSS v4**, **Framer Motion**, and **React Icons**.

The website is designed for **Kaustubh Dattatray Shinde** to showcase engineering, Java Full Stack, React, and AI/Embedded systems expertise in a clean, modern glassmorphic look.

---

## ✨ Features

- **Premium Aesthetics**: Glassmorphic panels, deep dark theme (default), glowing ambient backdrops, and active scroll indicators.
- **Dynamic Backgrounds**: High-performance, canvas-driven interactive particle networks.
- **Scroll & Spotlight Effects**: Ambient glowing spotlight following the cursor, scroll progress bar, and floating scroll-to-top button.
- **Filterable Projects**: Seamless project grouping tabs (All, Full Stack, AI & Embedded, Frontend) and details modal overlays.
- **Scientific Capstones Deep Dive**: Interactive tab layouts breaking down capstone systems (System Architecture, Speech Recognition, OpenCV Vision, and Gemini AI).
- **Interactive Contact Form**: Complete state validation check validating credentials and emails with active submission states.
- **SEO & Accessiblity Ready**: Comprehensive description meta headers, Open Graph blocks, `robots.txt`, and XML sitemaps.
- **Visitor Counter**: Persistent session-sensitive visitor telemetry.

## 🛠️ Quick Start

Follow these steps to run the portfolio locally on your machine:

### 1. Install Dependencies
Run the package installations from the root folder:
```bash
npm install
```

### 2. Launch Development Server
Start the local Vite preview server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.

### 3. Production Build
Verify standard compilation and bundle code sizes:
```bash
npm run build
```

## 🚀 Deployment Guide

### Vercel Deployment
1. Install Vercel CLI or import the repository directly on the [Vercel Dashboard](https://vercel.com).
2. Configure settings:
   - **Framework Preset**: `Vite`
   - **Build Command**: `npm run build` or `vite build`
   - **Output Directory**: `dist`
3. Click **Deploy**.

### Netlify Deployment
1. Install Netlify CLI or import the repository on the [Netlify Dashboard](https://netlify.com).
2. Configure build configurations:
   - **Build command**: `npm run build`
   - **Publish directory**: `dist`
3. Click **Deploy**.
